## 文本编辑器

    用户通过入口（url地址），输入文件地址（url参数），打开此编辑器编辑此文件，并进行后续的修改和保存。
    同时提供对此文件内容编辑的一系列功能。

**架构**

    纯前端架构，采用 angular+meterial实现。对于文件等内容获取和保存，采用coresdk实现。
    文本编辑器内核使用ace，核心编辑操作用此来提供。

**功能**

    文件
        保存
        退出
    编辑
        查找
        替换
    查看
        放大
        缩小
        还原默认大小
    设置
        主题
        字体
        自动换行


**--**主要流程
1通过url，传入文件地址，在编辑器中打开当前文件。
2修改文件
3保存更改
